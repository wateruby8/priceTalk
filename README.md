# 價情通報局
一款讓使用者也可以參與提供資料的比價系統

## 步驟一：Project Setup

```sh
npm install
```

### 步驟二：Compile and Hot-Reload for Development

```sh
npm run dev
```

### 步驟三：找到對應的檔案做開發
+ 首頁區塊 src -> components -> vue檔案
+ 分頁 src -> views -> vue檔案

#### vue檔案-架構介紹
+ template = html架構
+ style = css美化
+ script = js互動

### Compile and Minify for Production

```sh
npm run build
```

### Lint with [ESLint](https://eslint.org/)

```sh
npm run lint
```
